1. No sistema, criar uma nova seção no menu do professor chamada **"Registrar Aluno"**, onde o professor poderá acessar o formulário de cadastro.

2. O formulário deve ser dividido em seções claras:
  - Dados Pessoais (Nome, data nascimento, gênero, cor/raça, nacionalidade, naturalidade, CPF, RG, endereço completo, quantidade de pessoas na casa)
  - Dados da Família (nome, telefone e grau de parentesco dos responsáveis, estado civil dos pais, responsável legal, contato de emergência)
  - Dados Escolares (turma, turno, ano de ingresso, escola, tipo de escola, repetência, notas recentes, necessidades especiais e laudos)
  - Dados de Saúde (condições médicas, medicações, alergias, tipo sanguíneo, cartão SUS)
  - Benefícios Sociais (Bolsa Família, NIS, outros programas, renda familiar em faixas)
  - Observações Gerais (campo aberto para observações)

3. Os campos obrigatórios devem ser validados (ex: nome do aluno, data nascimento, telefone do responsável).

4. Deve haver upload de arquivos para anexar laudos médicos ou boletins escolares.

5. Deve conter botões "Salvar Cadastro" e "Cancelar / Voltar".

6. O formulário deve ser responsivo para desktop e mobile.

7. Ao salvar, validar os campos obrigatórios e exibir mensagens de erro claras caso algo esteja faltando.

8. Os campos opcionais não devem impedir o salvamento.

9. Formate as perguntas e campos de forma amigável e acessível, com placeholders e seleções para gêneros, cor/raça (conforme IBGE), estado civil, tipo de escola, faixa de renda, etc.

10. Permita seleção para perguntas tipo sim/não, e campos condicionais (exemplo: se “Já repetiu de ano?” for sim, mostrar campo para qual ano).

11. Para endereço, valide CEP e permita preenchimento separado por rua, número, bairro, cidade, estado.
