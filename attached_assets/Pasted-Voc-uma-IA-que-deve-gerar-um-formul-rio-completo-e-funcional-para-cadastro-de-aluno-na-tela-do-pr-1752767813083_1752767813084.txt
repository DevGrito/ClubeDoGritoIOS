Você é uma IA que deve gerar um formulário completo e funcional para cadastro de aluno na tela do professor, com as seguintes especificações:

1. O formulário deve ser dividido em seções claras:
  - Dados Pessoais (Nome, data nascimento, gênero, cor/raça, nacionalidade, naturalidade, CPF, RG, endereço completo, quantidade de pessoas na casa)
  - Dados da Família (nome, telefone e grau de parentesco dos responsáveis, estado civil dos pais, responsável legal, contato de emergência)
  - Dados Escolares (turma, turno, ano de ingresso, escola, tipo de escola, repetência, notas recentes, necessidades especiais e laudos)
  - Dados de Saúde (condições médicas, medicações, alergias, tipo sanguíneo, cartão SUS)
  - Benefícios Sociais (Bolsa Família, NIS, outros programas, renda familiar em faixas)
  - Observações Gerais (campo aberto para observações)

2. Os campos obrigatórios devem ser validados (ex: nome do aluno, data nascimento, telefone do responsável).

3. Deve haver upload de arquivos para anexar laudos médicos ou boletins escolares.

4. Deve conter botões "Salvar Cadastro" e "Cancelar / Voltar".

5. O formulário deve ser responsivo para desktop e mobile.

6. Ao salvar, validar os campos obrigatórios e exibir mensagens de erro claras caso algo esteja faltando.

7. Os campos opcionais não devem impedir o salvamento.

8. Formate as perguntas e campos de forma amigável e acessível, com placeholders e seleções para gêneros, cor/raça (conforme IBGE), estado civil, tipo de escola, faixa de renda, etc.

9. Permita seleção para perguntas tipo sim/não, e campos condicionais (exemplo: se “Já repetiu de ano?” for sim, mostrar campo para qual ano).

10. Para endereço, valide CEP e permita preenchimento separado por rua, número, bairro, cidade, estado.

---

Construa o formulário com base nessas especificações, pronto para integração em um sistema educacional.

---

**Exemplo de campos e validações no formulário:**

- Nome Completo (input texto) – obrigatório
- Data de Nascimento (input data) – obrigatório, deve ser menor que hoje
- Gênero (select: masculino, feminino, outro, prefiro não dizer)
- Cor/Raça (select conforme IBGE: branca, preta, parda, amarela, indígena)
- CPF (input texto, validar formato)
- Telefone do responsável (input telefone) – obrigatório
- Upload de arquivo (botão para anexar arquivo)
- Botão salvar com validação
